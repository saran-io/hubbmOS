    Democratizing Automation With the AI-Powered Filter Assistant and Seamless Python Scripting

    Kaj Wierda

    5 min read | June 23, 2025

    Highlights

    • See how Automic Automation offers an AI-powered Filter Assistant that simplifies search and filtering.
    • Discover how Automic now offers a seamless Python scripting experience that requires no manual configuration.
    • Leverage the solution to make powerful capabilities more intuitive and accessible, democratizing automation.

    In the evolving landscape of enterprise automation, making powerful functionality accessible to a broader user base and streamlining operational processes are key priorities. At Broadcom, we are continually enhancing Automic Automation to ensure it remains an intuitive, efficient, and adaptable platform.

    That's why we're introducing these notable enhancements in Automic SaaS 24.5:

    • An AI-powered Filter Assistant for intuitive conversational filtering and sorting.
    • A seamless, out-of-the-box Python scripting experience that offers users more flexibility in how they build their automation use cases.  

    These updates are designed to simplify your daily operations and empower more users across your organization.

    1. The Automic AI Filter Assistant: Conversational efficiency for all

    Hot on the heels of powerful new AI-driven features in Automic 24.4, such as ASK_AI, we’ve enhanced the Automic Web Interface’s (AWI) robust search capabilities in Automic SaaS 24.5. This release introduces natural language processing (NLP), delivered via the AI Filter Assistant.

    Simplify your search

    The AI Filter Assistant allows you to interact with your automation environment using conversational language. Instead of constructing detailed manual filters, you can simply articulate your needs, and the assistant will translate your request into precise filter definitions for both object and task searches.

    This conversational approach is particularly beneficial for new Automic users, helping reduce the learning curve and enabling them to perform advanced searches and quickly gain relevant insights. Plus, for experienced users, this assistant streamlines the creation of sophisticated, multi-layered filters, which can speed up troubleshooting and operational tasks.

    Dynamic and intuitive interactions

    Whether you need to quickly locate VARA objects modified in the last three days, starting with "UC_," for example, or investigate tasks that ended in an error status over the past week (including deactivated ones), the AI Filter Assistant can help. You can also refine your results with subsequent prompts, such as "order by last modified ascending" or "list commented tasks only," fostering a more natural interaction. Filters you create can be saved for convenient future access.

    This feature aligns with Automic's broader strategy to use generative AI to:

    • Democratize automation: Making powerful Automic functionality accessible to a wider range of users, potentially reducing specialized training needs.
    • Boost efficiency: Supporting routine tasks and accelerating troubleshooting.
    • Enhance resilience: Speeding the delivery of insights that can aid proactive problem-solving.
    • Drive innovation: Exploring intuitive ways AI can simplify and enhance enterprise automation.

    2. Effortless Python scripting: Expand your automation capabilities with no configuration

    Python is recognized for its versatility in data science, AI and machine learning, and complex data processing. While Automic has long supported Python scripting as part of its enterprise-grade orchestration, this support previously required manual configuration.

    With Automic SaaS 24.5, we're introducing a truly seamless Python scripting experience that requires no manual configuration steps.

    Leverage Python's flexibility with Automic's robust control

    The “no configuration required" aspect means the Python runtime is ready to use out of the box. You can simply select the Python job type in AWI, paste your code, and execute it directly. This simplification provides notable advantages for various roles.

    For Automic users and managers:

    • Expand your automation horizon: Gain effortless access to Python's extensive ecosystem of libraries (covering AI and machine learning, data processing, and cloud APIs), broadening the scope of what you can automate and increasing efficiency.
    • Integrate Python-savvy teams: Enable your Python developers and data engineers to contribute to and manage workflows directly within Automic, promoting collaboration and faster development cycles.
    • Streamline management: Reduce setup times and eliminate the need for manual Python configuration, simplifying Python code deployment and execution.

    For data engineers and data scientists:

    • Transition scripts to enterprise workflows: Convert standalone Python scripts into reliable, scalable, and manageable data pipelines, utilizing Automic's capabilities for scheduling, monitoring, and error handling.
    • Orchestrate beyond individual jobs: Seamlessly integrate Python-driven tasks into broader IT and business processes, allowing Automic to act as a centralized orchestrator for complex cross-platform dependencies.
    • Utilize existing skills, reduce learning curve: Apply your Python expertise directly within Automic, focusing on the data and logic rather than infrastructure setup.

    Advancing automation for everyone

    Automic SaaS 24.5 demonstrates our continued commitment to making enterprise automation more intuitive and accessible. By incorporating AI-powered filtering and providing a straightforward path for Python development, we are making powerful automation more user friendly and efficient across your organization.

    These capabilities are available now with Automic SaaS 24.5 and will be part of the next major on-premises release, 26.0, in the first half of 2026. See the Automic SaaS 24.5 release announcement for more details.
